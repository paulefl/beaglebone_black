# Security Policy

## Unterstützte Versionen

| Version | Support |
|---|---|
| 1.x | ✅ Aktiv |
| < 1.0 | ❌ Kein Support |

## Sicherheitslücke melden

**Bitte melde Sicherheitslücken NICHT als öffentliches Issue!**

Sende eine E-Mail an: security@example.com

Bitte folgende Informationen angeben:
- Beschreibung der Lücke
- Betroffene Komponente und Version
- Schritte zur Reproduktion
- Mögliche Auswirkungen

Wir antworten innerhalb von 48 Stunden.
