# Chapter 9: BeagleBone Black Mechanical

> Source: https://docs.beagle.cc/boards/beaglebone/black/ch09.html

## Dimensions and Weight

| Spec | Value |
|------|-------|
| Size | 3.5" × 2.15" (86.36mm × 53.34mm) |
| Maximum height | 0.187" (4.76mm) |
| PCB Layers | 6 |
| PCB thickness | 0.062" |
| RoHS Compliant | Yes |
| Weight | 1.4 oz |
